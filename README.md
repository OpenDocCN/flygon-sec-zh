## 下载

### Docker

```
docker pull apachecn0/freelearn-sec-zh
docker run -tid -p <port>:80 apachecn0/freelearn-sec-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-sec-zh
freelearn-sec-zh <port>
# 访问 http://localhost:{port} 查看文档
```
